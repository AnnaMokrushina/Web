Пособие по установке демонстрационного проекта

Для тестирования проекта у вас в системе должны быть установлены IIS, .NET Framework 1.1, MS SQL Server любой редакции (например, MSDE). Также желательно, чтобы были установлены Client tools for MS SQL Server и Visual Studio .NET 2003.

1. Создайте папку с произвольным названием в директории InetPub/wwwroot и скопируйте туда файлы проекта.

2. Создайте виртуальный псевдоним для этой папки - forum.

3. При наличии установленных client tools загрузите скрипт (forumdb.sql) в Query Analizer и создайте базу данных.

4. Пропишите в файле web.config из числа файлов проекта логин и пароль администратора базы в разделе AppSettings в ключе ConnectionString.

5. После этого вы сможете загрузить проект через браузер, набрав в адресной сроке http://localhost/forum

6. Для загрузки проекта в Visual Studio .NET 2003 вы должны создать solution для web-проекта и подредактировать сам файл солюшина, указав в нем путь к виртуальной папке проекта (http://localhost/forum).
